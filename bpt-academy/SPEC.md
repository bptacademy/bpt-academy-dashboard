# BPT Academy Mobile App — Specification

## 1. Overview

**Purpose:** Build a dedicated mobile app for BPT Academy to manage student registration, program enrolment, progress tracking, multimedia training content, and coach–student communication.

**Key Objectives:**
- Centralise all student and program data
- Offer personalised progress dashboards
- Host and stream training session clips
- Deliver real-time notifications and announcements from coaches and the Academy

---

## 2. User Roles & Permissions

| Role | Capabilities |
|---|---|
| **Student** | Register & login, view enrolled programs, track progress, watch session clips, receive notifications, message coaches |
| **Coach/Admin** | Create/update student profiles, design programs & sessions, upload video clips, send notifications and announcements, view student progress dashboards |
| **Parent/Guardian** *(optional)* | View child's progress, receive notifications, pay fees |

---

## 3. Core Features

### 3.1 Student & Program Management
- **Onboarding flow:** Sign-up via email, mobile number, or social login
- **Profile:** Personal details, skill level (Beginner / Intermediate / Advanced / Competition)
- **Program catalog:** List of available programs by level; description, duration, schedule
- **Enrollment:** Students register for multiple programs; waitlist management

### 3.2 Progress Tracking
- **Milestones & Modules:** Track module completion and scores
- **Visual Dashboard:** Progress bars per program and overall skill rating
- **History:** Past sessions, test results, competition outcomes

### 3.3 Multimedia Training Library
- **Session Clips:** Coaches upload videos tagged by date, drill type, skill focus
- **Bookmarking & Comments:** Students bookmark favorite drills and leave comments/questions
- **Download & Offline:** Option to cache clips for offline viewing

### 3.4 Notifications & Communication
- **Push Notifications:** Session reminders, new content alerts, program updates
- **In-App Messaging:** One-on-one chat between coach and student, group announcements
- **Email/SMS Integration:** Optional fallback channel for critical notices

### 3.5 Analytics & Reporting
- **Coach Dashboard:** Overview of all student progress, identify drop-off points
- **Engagement Metrics:** App usage stats, most viewed clips, active students per program

---

## 6. User Stories

- **As a student,** I want to see my program's progress dashboard so that I know what I've completed and what's next.
- **As a coach,** I want to upload session videos quickly so that students can review drills anytime.
- **As a parent,** I want to receive notifications if my child fails to complete a module.
